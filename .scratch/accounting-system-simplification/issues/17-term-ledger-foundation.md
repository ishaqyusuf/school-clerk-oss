# 17 — Term Ledger Foundation

**What to build:** Establish Term Ledger as the operator-facing financial record for an academic term, with lifecycle basics and a minimal current-term overview.

**Blocked by:** None — can start immediately

**Status:** ready-for-agent

- [ ] Each academic term can be represented as a Term Ledger.
- [ ] Term Ledger statuses support the approved lifecycle: Draft, Open, Closing, Closed, and Reopened.
- [ ] The current active term can load a minimal ledger overview.
- [ ] Closed terms remain readable and stable.
- [ ] Admin/Accountant access follows current finance-read permission expectations.
- [ ] Tests cover ledger creation/lookup, status transitions allowed in this slice, tenant boundaries, and current-term overview loading.
